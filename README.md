## 📖 项目简介

本项目是第六章的数据可视化练习，专注于Matplotlib坐标轴的高级定制技术。通过5个精心设计的交互式练习，帮助学习者掌握数据可视化的核心技能。

## 📋 练习内容

### 🎨 练习1：定制刻度位置、格式与样式
- **主题**：48小时股票价格走势图
- **技能点**：自定义刻度、格式化标签、样式定制
- **特色功能**：实时数据更新 + 手动数据输入

### 📊 练习2：添加图内坐标轴
- **主题**：2024年销量数据分析
- **技能点**：嵌入式子图、主图协调、图内布局
- **特色功能**：主图+嵌入图双数据输入

### 📈 练习3：隐藏部分轴脊
- **主题**：编程语言受欢迎程度投票
- **技能点**：轴脊显示隐藏、条形图定制、数据排序
- **特色功能**：动态排序 + 投票数据模拟

### 📐 练习4：移动轴脊到数据中心
- **主题**：二次函数 y = x²
- **技能点**：轴脊位置调整、中心化坐标系
- **特色功能**：系数调节 + 函数实时更新

### 🔍 练习5：综合定制（轴脊+刻度）
- **主题**：三角函数可视化
- **技能点**：不连续性显示、多函数切换、渐近线绘制
- **特色功能**：5种三角函数 + 垂直渐近线正确显示
