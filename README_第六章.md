## 练习内容

本章主要练习Matplotlib坐标轴的高级定制，包括以下五个练习：

### 练习1：定制刻度位置、格式与样式
- **内容**: 48小时股票价格走势图
- **技能**: 
  - 自定义刻度位置
  - 格式化刻度标签
  - 样式定制
- **交互功能**: 
  - 更新股票价格数据
  - 生成随机价格数据
  - 重置数据
  - **手动数据输入**

### 练习2：添加图内坐标轴
- **内容**: 2024年销量数据分析（主图+嵌入图）
- **技能**:
  - 创建嵌入式子图
  - 主图与子图协调
  - 图内坐标轴布局
- **交互功能**:
  - 更新全国销量数据
  - 更新华东区数据
  - 交换数据
  - **主图和嵌入图分别支持手动数据输入**

### 练习3：隐藏部分轴脊
- **内容**: 编程语言受欢迎程度投票结果
- **技能**:
  - 轴脊的显示与隐藏
  - 条形图样式定制
  - 数据排序
- **交互功能**:
  - 更新投票数据
  - 随机投票数据
  - 数据排序
  - 重置数据
  - **手动数据输入**

### 练习4：移动轴脊到数据中心
- **内容**: 二次函数 y = x²
- **技能**:
  - 轴脊位置调整
  - 中心化坐标系
  - 函数图像绘制
- **交互功能**:
  - 更新二次函数
  - 随机系数
  - 系数调节
  - **手动数据输入**

### 练习5：综合定制（轴脊+刻度）
- **内容**: 三角函数可视化
- **技能**:
  - 正切函数不连续性正确显示
  - 多种三角函数切换
  - 垂直渐近线绘制
  - 坐标轴范围动态调整
- **交互功能**:
  - 函数类型切换（正切、正弦、余弦、余切、正割）
  - Y轴范围调节
  - **手动数据输入**
  - **修复了正切函数不连续性问题**

## 重要修复

### 正切函数不连续性问题修复
- **问题**: 原始代码将正切函数显示为连续曲线
- **解决方案**: 
  - 将函数分段绘制，正确显示间断点
  - 添加垂直渐近线（虚线）
  - 优化坐标轴标签显示

### 手动数据输入功能
- **新增功能**: 所有练习都支持手动数据输入
- **实现方式**: 每个练习都添加了输入框和应用按钮
- **数据验证**: 包含完整的错误处理和数据验证
- **调试功能**: 添加了详细的控制台日志输出
